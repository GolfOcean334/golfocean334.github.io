Chess Game

Présentation :

Chess Game est un jeu d’échecs 2D développé en C++ avec SFML, proposant deux modes : Joueur vs Joueur et Joueur vs IA (algorithme MinMax).

Modes de jeu :

  • Joueur vs Joueur — deux joueurs se partagent le même ordinateur.
  • Joueur vs IA — le joueur joue les blancs contre une IA (MinMax) jouant les noirs.

IA MinMax :

L’IA évalue les positions via l’algorithme MinMax pour choisir le meilleur coup selon une profondeur donnée.

Fonctionnalités principales :

  • Multijoueur local (tour par tour sur le même PC).
  • IA avec prise de décision via MinMax.
  • Rendu 2D avec SFML.
  • Affichage clair de l’échiquier et des pièces.
  • Gestion du tour par tour et des déplacements valides.
